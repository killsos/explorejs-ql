About this book

This book is about two versions of JavaScript:

  • ECMAScript 2018 and
  • ECMAScript 2019

It only covers what’s new in those versions. For information on other versions, consult my other books, which are free to read online, at ExploringJS.com.

This book isn’t finished, yet

“Exploring ES2018 and ES2019” is a living book. More content is still to come:

  • ES2018: The feature set of ES2018 has been finalized and is described in this book.
  • ES2019: is work in progress. I track potential feature set in a blog post. Once a feature is confirmed for ES2019, it will be documented in this book.
