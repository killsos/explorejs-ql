Exploring JS: JavaScript books for programmers

All of the following books are free to read online. I hope you’ll like them so much that you’ll buy the offline versions.
— Dr. Axel Rauschmayer (author of the 2ality blog)

The books

  • Speaking JavaScript
    • JavaScript for programmers
    • No required knowledge (apart from programming)
    • Versions of JavaScript: everything, up to and including ECMAScript 5 (ES5)
  • Exploring ES6
    • Version of JavaScript: ECMAScript 6 (also known as ES2015)
  • Exploring ES2016 and ES2017
    • Versions of JavaScript: ECMAScript 2016 and ECMAScript 2017
  • Exploring ES2018 and ES2019
    • Versions of JavaScript: ECMAScript 2018 and ECMAScript 2019
  • Setting up ES6
    • Using and configuring Babel 6 in web browsers and on Node.js.

About the author

Dr. Axel Rauschmayer specializes in JavaScript and web development. He blogs, writes books and teaches classes.

Axel has been writing about the future of JavaScript since early 2011.
